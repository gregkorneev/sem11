Проект демонстрирует умножение квадратных матриц двумя способами:
```
Стандартный алгоритм с тройным циклом O(n^3) .
```
```
Алгоритм Штрассена с асимптотикой O(n^{log₂7}) ≈ O(n^{2.807}) ,
```
с выводом промежуточных матриц M1–M7 и блоков C11, C12, C21, C22 .
```
Размер матриц — n x n , где n должно быть степенью двойки (2, 4, 8, 16, ...).
```

Кратко о реализованных алгоритмах
1. Стандартное умножение
Тройной цикл:
```
for (int i = 0; i < n; ++i)
```
```
for (int j = 0; j < n; ++j) {
```
```
double sum = 0.0;
```
```
for (int k = 0; k < n; ++k)
```
```
sum += A[i][k] * B[k][j];
```
```
C[i][j] = sum;
```
```
}
```
Число операций ~ n³ .
```
Асимптотика: O(n^3) .
```
2. Алгоритм Штрассена
Матрицы разбиваются на 4 блока n/2 x n/2 : A11, A12, A21, A22 и
B11, B12, B21, B22 .
Вычисляются 7 промежуточных матриц M1..M7 .
Итоговые блоки C11..C22 собираются из комбинаций M1..M7 .
Рекуррентная оценка сложности:
```
T(n) = 7 * T(n/2) + O(n^2)
```
Отсюда:
```
T(n) = O(n^{log₂7}) ≈ O(n^{2.807})
```
